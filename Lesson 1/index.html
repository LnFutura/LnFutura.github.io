Примение CSS

CSS – это формальный язык, служащий для описания оформления внешнего вида документа, созданного с использованием языка разметки (HTML, XHTML, XML). Название происходит от английского Cascading Style Sheets, что означает «каскадные таблицы стилей». Назначение CSS – отделять то, что задает внешний вид страницы, от ее содержания. Если документ создан только с использованием HTML, то в нем определяется не только каждый элемент, но и способ его отображения (цвет, шрифт, положение блока и т. д.). Если же подключены каскадные таблицы стилей, то HTML описывает только очередность объектов. А за все их свойства отвечает CSS. В HTML достаточно прописывать класс, не перечисляя все стили каждый раз.

  • Идентификатор задает уникальное имя элемента.
  • Идентификаторы уникальны для одной страницы.
  • Каждый элемент может иметь только один id, также как и id уникален для страницы.
  • Классы применяют, когда нужно задать стиль одного элемента или задать разные стили для одного тега.
  • Можно использовать одинаковые классы для разных элементов или же использовать разные классы для одинаковых элементов, тоесть миксовать.
